# PRIME ACADEMY

Landing page for PRIME ACADEMY, an online physics school for OGE, EGE, olympiad preparation, and school physics.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- lucide-react

## Run Locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite, usually:

```text
http://localhost:5173/
```

## Build And Check

```bash
npm run build
npm run lint
```

## Images

Teacher and course photos are imported from `src/assets/images` so Vite can fingerprint and optimize the final build assets.
